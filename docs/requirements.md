# Project Requirements

## System Requirements
- Node.js >= 18.0.0
- npm >= 9.0.0
- Bash >= 4.0
- Docker >= 24.0.0
- Docker Compose >= 2.0.0

## Development Tools
- TypeScript >= 5.0.0
- Vite >= 6.0.0
- Vitest >= 1.0.0
- ESLint >= 8.0.0
- Prettier >= 3.0.0

## Runtime Dependencies
- PostgreSQL >= 16.0
- Redis >= 7.0

## Environment Variables
- DATABASE_URL
- PORT
- NODE_ENV
- POSTGRES_USER
- POSTGRES_PASSWORD
- POSTGRES_DB
- REDIS_PORT
